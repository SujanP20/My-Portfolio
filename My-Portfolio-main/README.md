# 🚀 Surya Kiran N — Portfolio Website

A modern, dark-themed portfolio website built with **HTML**, **CSS**, and **JavaScript**, featuring smooth GSAP animations, scroll-triggered effects, and a premium design aesthetic.

## 🌟 Live Preview

Open `index.html` in your browser to view the portfolio.

## ✨ Features

- **Hero Section** — Animated flowing cards with infinite scroll
- **Tech Stack Marquee** — Auto-scrolling technology logos (Python, Java, TensorFlow, etc.)
- **Featured Projects** — Horizontal scroll gallery with click-to-view project modals
- **Technical Skills** — Interactive cards with hover image reveal
- **Education** — Grid layout with clickable Certifications popup showing real certificates
- **About Me** — Professional summary with animated stats
- **Let's Connect** — Contact cards with real brand SVG icons (LinkedIn, GitHub, Email, Phone)
- **Skills Marquee** — Infinite text scroll with animated separators
- **Side Menu** — Full-screen overlay navigation with smooth GSAP transitions
- **Custom Cursors** — Context-aware cursor effects on project cards

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Structure & semantic markup |
| CSS3 | Styling, animations, responsive design |
| JavaScript | Interactivity & DOM manipulation |
| GSAP + ScrollTrigger | Smooth scroll animations & pinning |
| Google Fonts | Inter & Syne typography |
